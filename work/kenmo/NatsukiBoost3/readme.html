・はじめに
  ７年の時を経てあのゲームが帰ってきた！
  菜月ブースト３ここに見参！

・ストーリー
  宇宙世紀3022年…
  宇宙おやつ襲来から７年。
  菜月の活躍により宇宙に平和がもたらされ、同時に食料事情も解決した。

  か、に見えた！
  奴らは銀河の彼方で宇宙侵略の機会をうかがっていたのである。

  視界一面に広がる「宇宙ウエハース」の大群！
  すべてを飲み込みながら迫る宇宙ウエハースを前に再び少女が立ち上がる。

  ウエハースは鮮度が命。
  いかに早く捕獲できるかが作戦の鍵を握る。
  人類は食われる前に食うことができるのか…

・操作方法
  マウス：キャラクターの上下移動
  左クリック：ブレーキ

  Kenmo
